Custom Workout Timer C | JS | HTML | CSS | PHP App Store | Source
This Pebble app allows you to easily create custom interval timers (with names) for your workouts.The Pebble will time you and alert you when it's time to change moves based on your own workout program. Example: Plank for 2 minutes, pushups for 45 seconds... etc.
The app consists of a Pebble side C code that request moves from the phone app (writen in JS). The online workout manager uses a combination of HTML/CSS, JS and PHP to allow users to enter workouts from their phone or computer.
There are currently over 4000 people using this app in more than 100 countries around the world.
